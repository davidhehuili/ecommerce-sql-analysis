# E-commerce Data Analysis using SQL

Author: David Li

## Project Overview
This project analyzes an e-commerce dataset using SQL to understand revenue performance, customer behavior, and product trends.

## Business Questions
- What is the total revenue and total number of orders?
- How does revenue change month by month?
- What is the cancellation rate?
- Who are the highest value customers?
- What product categories perform best?

## Tools Used
SQL (MySQL)

## Key Insights
- Total revenue is approximately 14.4 million.
- The cancellation rate is around 0.63%.
- The top 10% of customers contribute about 41.65% of total revenue.
- The product category "beleza_saude" shows strong market demand.
